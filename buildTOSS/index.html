Data journalist and developer

Building and using tools to tell stories.

I analyze and extract key points from data to create maps and charts and meaningful interactives. Sometimes I build bots that gather information and others that can chat. I write code using Javascript, Python and R. Check out some of my work. Occasionally I teach or talk about data and visual journalism. I also contribute to a few open source projects.
